セールスフォスOauthフローでMCAE API用のToken取得

1.Core組織にログインしてコードを取得：

Connect with Pardot

2. Responseで取得したCode　を使って、Tokenを取得

例： 

curl --location 'https://login.salesforce.com/services/oauth2/token' \
--header 'Authorization: Basic Og==' \
--header 'Cookie: BrowserId=GvrIjK_wEfCk40GaSQuDmg; CookieConsentPolicy=0:0; LSKey-c$CookieConsentPolicy=0:0' \
--form 'grant_type="authorization_code"' \
--form 'client_id=＜CLIENTID＞' \　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　＜＜＜＜接続App設定から取得
--form 'client_secret=' \　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　＜＜＜＜接続App設定から取得
--form 'code="aPrxKlcUBVrDN50Lcwmbt2c_WTvm0Z47JjCR7OcQs.GAU0rreGve_qSV22.SsLHquJ7BEGAWOA=="' \    ＜＜＜ ここに入れる
--form 'redirect_uri="https://heroku.yukikolocal.com/Pardot_API.html"'　　　　　　　　　　　　　　　　＜＜＜＜接続Appで指定しているやつ
